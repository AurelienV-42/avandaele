Agence Minuit is a specialized agency in mobile application development.

During this assignment, I had the opportunity to work on two application projects:

- The first was the complete development of a mobile application with "complex" modules such as an interactive map, swipeable cards (Tinder-like), chat management, and notifications.

- The second required two developments: a simple mobile application and a web application for another user profile. Conceptualizing the architecture was very challenging because the goal was to avoid code duplication between the web and mobile parts. The various components of these applications included an interactive map (Uber-like, moving a car on a map) and rapid matchmaking (direct contact with the client...).
