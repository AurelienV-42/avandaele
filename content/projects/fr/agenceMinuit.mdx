Agence Minuit est une agence spécialisée dans le développement d'applications mobiles.

Durant cette mission, j'ai eu l'opportunité de travailler sur deux projets d'applications :

- Le premier était le développement complet d'une application mobile avec des modules "complexes" tels qu'une carte interactive, des cartes swipables (type Tinder), gestion de chat et notifications.

- Le second nécessitait deux développements : une application mobile simple et une application web pour un autre profil utilisateur. Conceptualiser l'architecture était très challengeant car l'objectif était d'éviter la duplication de code entre les parties web et mobile. Les différents composants de ces applications incluaient une carte interactive (type Uber, déplacement d'une voiture sur une carte) et un matching rapide (contact direct avec le client...).
