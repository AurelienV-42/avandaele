J'ai accompagné Skin Easy, un institut de soins du visage, dans la conception et le développement complet de leur nouvelle application mobile, connectant une infrastructure moderne à leur plateforme e-commerce existante (PrestaShop).

Cette mission a permis à Skin Easy de :

**Développement Backend** : Conception d'une API robuste avec NestJS & Prisma, traduisant des règles métier complexes (diagnostic de peau, incompatibilités d'ingrédients) pour générer des routines de soins personnalisées.

**Création d'une expérience unifiée (React Native)** : Développement de l'application mobile (iOS/Android via Expo) incluant un journal de suivi (Sommeil, Alimentation, Peau). Utilisation de React Native Web pour rendre une page critique disponible sur le site web avec la même base de code.

Cette mission a fourni à Skin Easy un outil de fidélisation propriétaire et évolutif, centralisant toute l'intelligence métier dans une seule API servant à la fois le Web et le Mobile.

**Stack technique** : React Native (Expo), React Native Web, NestJS, Prisma, MySQL, PrestaShop (Intégration).
