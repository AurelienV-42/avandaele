# Développeur Front - React Native

Circular est une start-up qui réalise des bagues connectées dotée de biocapteurs et connectée à une application qui analyse les signaux du corps, permettant d’améliorer le sommeil, les activités physiques et la santé.

Sur cette mission j'ai continué le développement de leur application en React Native. J'ai eu la possibilité d'aider Circular à mettre en production leur produit sur l'App Store.
